# Мониторинг можно разделить на следующие виды 
* **Графическое представление**
* **Нотификация**

# Инструменты мониторинга 
## Данные
Должно быть место для хранения метрик. 
Это может быть база данных
* классическая RDBMS 
* Аналитическая БД 
* TimeSeriesDB

Критерий 
Набор всевозможных метрик:
* Продолжительность формирования данных
* Успешность
* Статистические метрики о расходе ресурса
* Отставание данных
* Метрики качества данных
* Ошибки по причине неправильного конфигурирования
* Ошибки в связи со сбоями в инфраструктуре
